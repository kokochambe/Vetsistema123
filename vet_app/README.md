# Ветеринарная аптека - Система управления запасами и продажами

## Описание
Рабочая программа для управления запасами и продажами ветеринарных товаров.

## Требования
- Python 3.8+
- tkinter (обычно входит в стандартную поставку Python)

## Запуск программы

### Linux/Ubuntu
```bash
# Установка tkinter (если не установлен)
sudo apt-get install python3-tk

# Запуск программы
python3 vet_pharmacy.py
```

### Windows
```bash
python vet_pharmacy.py
```

### macOS
```bash
python3 vet_pharmacy.py
```

## Функционал

### 📦 Вкладка "Товары"
- Просмотр всех товаров в виде таблицы
- Добавление новых товаров
- Редактирование существующих товаров
- Удаление товаров
- Фильтрация по категориям
- Фильтр "Мало на складе" (товары с количеством ≤ 10)

### 🛒 Вкладка "Продажи"
- Выбор товара из доступных на складе
- Добавление товаров в корзину
- Изменение количества
- Удаление товаров из корзины
- Оформление продажи с указанием клиента
- Автоматическое списание товаров со склада
- Подсчет итоговой суммы

### 📊 Вкладка "Отчеты"
- **Продажи**: история всех продаж с датами и суммами
- **Мало на складе**: товары, требующие пополнения
- **Истекает срок**: товары с истекающим сроком годности

## Структура данных

Программа хранит данные в файле `vet_data.json`:
- Список товаров (название, категория, цена, количество, производитель, срок годности)
- История продаж

## Категории товаров
- Корма
- Лекарства
- Аксессуары
- Гигиена
- Витамины

## Единицы измерения
- шт (штуки)
- кг (килограммы)
- г (граммы)
- л (литры)
- мл (миллилитры)
- уп (упаковки)
- фл (флаконы)
- таб (таблетки)

## Инструкция по использованию

1. **Добавление товара**:
   - Перейдите на вкладку "Товары"
   - Нажмите "➕ Добавить"
   - Заполните все поля формы
   - Нажмите "Сохранить"

2. **Оформление продажи**:
   - Перейдите на вкладку "Продажи"
   - Выберите товар из списка
   - Укажите количество
   - Нажмите "➕ Добавить"
   - Повторите для других товаров
   - Укажите имя клиента (опционально)
   - Нажмите "💰 Оформить продажу"

3. **Просмотр отчетов**:
   - Перейдите на вкладку "Отчеты"
   - Переключайтесь между подвкладками для просмотра различной информации
   - Нажмите "🔄 Обновить" для актуализации данных
